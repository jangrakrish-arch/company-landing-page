# 🍔 QuickBite - Food Delivery Landing Page

## 📌 Project Description
This is a simple and responsive landing page for a fictional food delivery service called **QuickBite**.  
It showcases features, service areas, customer reviews, and contact details.

---

## 🚀 Features
- Attractive Hero Section
- Feature Highlights with Icons
- Customer Reviews Section
- Responsive Design (Mobile Friendly)
- Hover Effects for Better UI

---

## 🛠️ Technologies Used
- HTML5
- CSS3
